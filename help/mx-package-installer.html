HELP: MX Package Installer

This help is for the Arch Linux build.

Repositories

  • Search the official repositories (pacman) for packages.
  • Installed packages are shown when the search box is empty.
  • Use the status filter to view installed, upgradable, or not installed packages.

AUR

  • Search AUR packages using paru. Results appear after pressing Enter.
  • Packages are user-contributed and not reviewed; use with care.
  • Use the status filter to view installed, upgradable, or not installed packages.

Flatpaks

Flatpaks are self-contained packages that work across multiple distributions.

  • The first Flatpak install may download a large runtime.
  • The default remote is flathub; additional remotes can be added in Advanced.

Console Output

  • Command output appears in the Console Output tab.
